一、java

1.数据类型

  • byte/8
  • char/16
  • short/16
  • int/32
  • float/32
  • long/64
  • double/64
  • boolean
    • boolean无明确规定，但是在jvm编译过程期间，将boolean转换成int，使用0和1表示false和true，可以使用1bit存储。

2.缓存池

1).new Integer(123) 与 Integer.valueOf(123) 的区别：

new Integer(123)，创建了一个新的对象，Integer.valueOf(123) 则是先查一下缓存池，如果有缓存则直接返回，多次调用会返回一个对象的引用

Integer.valueOf()源码

各个类型的缓存池大小：

  • boolean values true and false
  • all byte values
  • short values between -128 and 127
  • int values between -128 and 127
  • char in the range \u0000 to \u007F

2).string常量池

  • jvm会加载所有的字符串，将这些字符串存到常量池中，所有对其中的字符串的引用都是同一个地址。这些字符串在编译时期就已经确定了。
  • 字符串拼接的时候

3.final关键字

  • 修饰常量
    • 常量不可变
  • 修饰变量
    • 变量的引用不可变，但是对象本身是可以修改的
  • 修饰方法
    • 方法不能被重写
    • 当使用private修饰方法的时候，private方法被隐式的指定了final修饰
  • 修饰类
    • 该类不能被继承

4.static关键字

5.transient

  • 被transient修饰的变量，不能被序列化

二、集合

1.ArrayList

  • 实现了RandomAccess接口，支持快速随机访问
  • 快速查找效率高，但是在插入或者删除效率低，因为实现了RandomAccess接口，所以查找效率高，在做插入或者删除操作的时候，会把index位置后面的数据复制到index处，使用的是系统底层的System.arraycopy()方法，时间复杂度是O(N)，代价是非常高的。
  • 默认初始化是10，扩容是原来的1.5倍
